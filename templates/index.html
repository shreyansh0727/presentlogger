System Architecture

How everything works together

Hardware Layer

  • Arduino UNO R4 WiFi microcontroller
  • MFRC522 RFID Reader (13.56 MHz)
  • 0.96" OLED Display (I2C)
  • PIR Motion Sensor
  • LED indicators and buzzer

Communication Layer

  • WiFi connection to Flask server
  • HTTPS RESTful API endpoints
  • WebSocket for real-time updates
  • API key authentication

Backend Layer

  • Flask web framework (Python 3.11)
  • MongoDB database with connection pooling
  • Flask-SocketIO for WebSocket
  • Email notifications with Flask-Mail

Frontend Layer

  • Responsive HTML5/CSS3 dashboard
  • Chart.js for data visualization
  • Real-time updates via Socket.IO client
  • Admin panel with CRUD operations
